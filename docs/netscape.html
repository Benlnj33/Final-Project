Netscape Overview

Netscape is a brand name associated with the development of the Netscape web browser. It is now owned by Oath, Inc., a subsidiary of Verizon. The brand belonged to the Netscape Communications Corporation (formerly Mosaic Communications Corporation), an independent American computer services company.

Netscape is credited with creating the JavaScript programming language, the most widely used language for client-side scripting of web pages. The company is also known for developing SSL which was used for securing online communications before its successor TLS took over.
